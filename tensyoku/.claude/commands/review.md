# 週次レビュー：運用全体を振り返る

毎週1回実行して、運用全体を振り返ります。

## ステップ1：全ファイルを確認する
- `post-history.md`（投稿記録と数値）
- `post-queue.md`（待機中の投稿）
- `analysis-latest.md`（最新分析）
- `next-topics.md`（来週のテーマ）
- `07_ng-rules.md`（ルール遵守チェック）

## ステップ2：週次チェックリストを実行する

以下をYES/NOで確認してください：

- [ ] 今週の投稿、誰の何を解決したか説明できるか
- [ ] 投稿が伸びなかった場合、原因を3つ挙げられるか
- [ ] 1週間後のテーマが最低5個あるか
- [ ] 反応が良かった投稿を横展開しているか
- [ ] 有益投稿：PR投稿 ＝ 7：3の比率を守れているか
- [ ] 1日3件以上の投稿をしていないか
- [ ] PR投稿の本文に直接URLを貼っていないか
- [ ] NGルール（誹謗中傷・断定表現・PR未表記）の違反はないか

## ステップ3：改善提案を出す

- 今週上手くいったこと（続けるべきこと）
- 今週上手くいかなかったこと（改善すべきこと）
- 来週試すべき新しいアプローチ1〜3個
