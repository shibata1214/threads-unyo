### 転職エージェント紹介投稿（#3・#7）の必須ルール

紹介できるエージェント：
- ナース人材バンク
- レバウェル看護（旧看護のお仕事）
- マイナビ看護師
- その他看護師転職エージェント全般

PR投稿の構成テンプレート：
```
（フック：転職を迷っている状況を描写）

転職を考えてた時に（エージェント名）を使ったんだけど、

（具体的な体験・担当者の対応・求人の質など）

子持ちのこと話したら（具体的なエピソード）

詳しくはプロフのリンクから🔗

#PR
```

**絶対に守ること：**
- Threads投稿本文には直接URLを貼らない（プロフリンク経由）
- 体験していないことを体験したように書かない
- 「絶対採用される」「確実に年収UP」等の断定は禁止

### 全投稿の必須条件
- 文字数：200〜350文字
- 口調：タメ語（〜だよ、〜だよね、〜してみて）
- 1行目：`05_writing.md` のフック型から必ず選ぶ
- 1行目を5案出して、一番強い1つを選ぶ（固有名詞か数字を入れる）
- 構成：フック→解説→アドバイスorエージェント紹介→CTA
- `07_ng-rules.md` の禁止事項を全て守る
- AI感のある不自然な表現は使わない（「〜と言えるでしょう」等は禁止）
- 同じネタ・同じフックの重複は禁止
- PR投稿は必ず「#PR」を明記する

### セルフチェック（全投稿必須）
各投稿を書いたら以下を確認してください：
- [ ] 1行目で手が止まるか？
- [ ] AI感のある表現がないか？
- [ ] 07_ng-rules.md に違反してないか？
- [ ] 文字数は200〜350文字か？
- [ ] PR投稿にPR表記があるか？
- [ ] 投稿本文に直接URLが入っていないか？
