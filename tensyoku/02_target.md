# ターゲット像

## メインターゲット
子育て中（未就学児〜小学生）の看護師で転職を考えている人

## ペルソナ
- 年齢：28〜38歳
- 職場：クリニック・デイサービス・病棟（夜勤に疲れた人も含む）
- 家族：子ども1〜2人（0〜8歳）
- 悩み：
  - 子どもが熱を出すたびに職場に申し訳なくて限界
  - 夜勤があると子どもとの時間が取れない
  - 転職したいけど子持ちでも採用してもらえるか不安
  - 転職エージェントって何を使えばいいかわからない
  - 病棟経験がなくてスキルに自信がない
  - 今の職場を辞めたいけど次が決まるか怖い
- 行動：Threadsで「看護師 転職」「ナース ママ 転職」等を検索する
- 求めてるもの：同じ境遇の人のリアルな体験談と、具体的な行動のヒント
