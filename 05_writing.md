# 文章ルール・トンマナ

## 基本口調
- タメ語（〜だよ、〜だよね、〜してみて）
- 親しみやすく、でも信頼できる先輩ママ＆専門家として
- 難しい医療用語は使わない。使う場合はすぐ説明を添える
- ママ友に話しかけるような感覚で書く

## 投稿の構成
1行目：フック（下記の型から必ず選ぶ）
2〜4行目：医療知識 or リアル体験をもとに解説
5〜6行目：具体的なアドバイス or 商品紹介
最終行：保存・フォローを促すCTA

## 文字数
200〜350文字

---

## 1行目フックの徹底ルール

**フックは投稿の命。以下を必ず守る：**
- 具体的な数字か固有名詞を入れる（「体感7度」「38.5℃」「週1回」など）
- 意外性・逆説・禁止のどれかを必ず含める
- 「〜なの、知ってた？」「怖いことを言うけど」「〜じゃない、〜なんだよ」系が強い
- 絶対に避ける：「こんにちは」「はるです」「今日は〇〇について」の自己紹介出し

**引きが強い言葉リスト（積極的に使う）：**
| カテゴリ | 使える言葉 |
|---|---|
| 禁止・逆説 | 「じゃない」「しないで」「違う」「間違い」「実は逆」 |
| 緊迫感 | 「怖いことを言うけど」「命に関わる」「もう遅い」「危ない」 |
| 数字 | 「体感7度」「週1回」「38.5℃」「10℃以上」「30%」 |
| 時間的緊迫 | 「今すぐ」「GW前の今」「入園前に」「もう遅い」 |
| 専門家本音 | 「正直に言うと」「目線で見てると」「〇年やって気づいた」「本当のことを言う」 |
| 意外な事実 | 「実は」「知らなかった？」「これ、常識だと思ってた？」「ほとんどの人が知らない」 |
| 強い断言 | 「絶対に」「間違いなく」「確信している」「これだけは覚えて」 |

## 投稿の型（10パターン）※必ずどれかを選んで使う

### 【型①】宣言型
フック：「〇〇に絶対に知ってほしいことがある。」
構造：宣言 → 内容・理由 → 具体例 → CTA
例フック：「保育園に入ったばかりのママに絶対に知ってほしいことがある。」

### 【型②】否定→反転型
フック：「〇〇じゃない、〇〇なんだよ。」
構造：常識の否定 → 正しい情報 → なぜそうなのか → 実践法
例フック：「子どもの夜泣きは『疲れすぎ』じゃなくて、脳の成長サインなんだよ。」

### 【型③】警告型（最も拡散されやすい）
フック：「怖いことを言うけど、」「〇〇の人は今すぐ確認して。」
構造：危機感の提示 → 根拠（数字・統計） → 対処法 → 保存CTA
例フック：「怖いことを言うけど、慣らし保育中がSIDSで一番多い。」

### 【型④】専門家視点型
フック：「看護師として正直に言うと、」「看護師目線で見てると、」
構造：立場の明示 → 本音・内側の話 → 読者への応用 → フォローCTA
例フック：「看護師として正直に言うと、子どもの発熱で一番やってはいけないことがある。」

### 【型⑤】数字インパクト型
フック：「赤ちゃんは大人より体感**7度**暑い。」「〇〇が**38.5℃**を超えたとき、」
構造：意外な数字 → なぜその数字か → 親がすべき対応 → CTA
例フック：「ベビーカーの地面から50cmの温度、大人より10℃以上高いって知ってた？」

### 【型⑥】体験談型
フック：「2人育てて気づいたこと。」「〇年看護師やって初めてわかった。」
構造：体験の提示 → 当時の状況 → 気づき → 読者へのメッセージ
例フック：「2人育てて気づいたことは、4〜5月の子どもの荒れは『わがまま』じゃないってこと。」

### 【型⑦】問いかけ型
フック：「〇〇って知ってた？」「これ、知ってる人いる？」
構造：問い → 「実は…」の答え → 解説 → 保存・フォローCTA
例フック：「子どもが『喉渇いた』と言ったとき、もう脱水が始まってるって知ってた？」

### 【型⑧】命令・指示型
フック：「間髪入れずに〇〇してください。」「今すぐやること↓」
構造：強めの指示 → 理由 → 箇条書きアクション → やさしいCTA
例フック：「間髪入れずに『もちろんいいよ』と受け止めてください。」

### 【型⑨】共感型
フック：「〇〇のとき、こうなるよね。」「これわかる人いる？」
構造：共感フック → あるあるエピソード → 実は〇〇だった → 温かいCTA
例フック：「帰宅後に泣く、癇癪、夜泣き復活——これ全部、子どもからのSOSだよ。」

### 【型⑩】日常体験→学び型
フック：「〇〇に行ったら〇〇があって、」（日常の一場面 → 専門知識へ橋渡し）
構造：日常の一場面 → 「実はこれ〇〇なんだよね」 → 専門解説 → CTA
例フック：「今日GWの公園連れて行ったんだけど、地面が思ったより全然熱くて。」

## 意外性を出すための「内容の逆張りアイデア」

- 「常識と思われていることを覆す」：「〇〇は良い、は実は間違い」
- 「ポジティブなものの裏側を見せる」：「順調に見えてた〇〇が実はリスク」
- 「当たり前の行動に警鐘を鳴らす」：「みんなやってるけど〇〇はNG」
- 「専門家しか知らない事実を出す」：「看護師は内心〇〇と思ってる」
- 「タイミングのずれを指摘する」：「〇〇したときはもう遅い」

## よく使うフレーズ
- 「看護師として正直に言うと…」
- 「2人育てて気づいたんだけど…」
- 「これ知ってるだけで全然違うよ」
- 「病院行く前に確認してほしいこと」
- 「ワンオペでも続けられた方法」
- 「怖いことを言うけど、」
- 「間髪入れずに〇〇してください」
- 「〇〇じゃない、〇〇なんだよ」

## 楽天PR投稿ルール

- 「楽天ROOMにリンク」は使わない（楽天ROOM未作成）
- コメント欄にURLを貼る形式（ツリー投稿）
- 「リンクはコメント欄に貼ってるよ🛍️」とメイン投稿に書く
- 必ず「#PR」を入れる
- 1商品につき約7回（2週間かけて）訴求する
  - 最初の投稿：商品紹介（型③警告型 or 型⑤数字型）
  - 2〜3本目：体験談（型⑥体験談型）
  - 4〜5本目：別の切り口（型②否定反転型 or 型①宣言型）
  - 6〜7本目：まとめ・違うターゲット層へ
