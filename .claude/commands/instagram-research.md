## ステップ2：ブラウザでInstagramを検索する

Instagramの検索 `https://www.instagram.com/explore/tags/{キーワード}/` にアクセスして、
再生数・いいね数が多いリールをピックアップしてURLを取得してください。

取得対象：
- リール（動画）
- バズっている通常投稿（画像・テキスト）
- キャプション（投稿文）の全文

## ステップ3：動画をダウンロードして文字起こしする

```bash
# yt-dlpのインストール（初回のみ）
pip install yt-dlp openai-whisper 2>/dev/null || true

# 動画ダウンロード
yt-dlp -o "/tmp/instagram_%(id)s.%(ext)s" {動画URL}

# Whisperで文字起こし
python3 -c "
import whisper
model = whisper.load_model('base')
result = model.transcribe('/tmp/instagram_{動画ID}.mp4', language='ja')
print(result['text'])
"
```

動画が取得できない場合は、キャプションと画像の文字情報から内容を分析してください。

## ステップ4：バズった投稿の構成を分析する

各投稿から以下を分析してください：

**フック分析（1行目の分析）：**
- どんな書き出しで始まっているか
- 数字・否定・疑問・感情ワードのどれを使っているか
- はるさんのアカウントで転用できるフック案を3つ抽出

**構成分析：**
- 投稿の型（箇条書き型・ストーリー型・断言型）
- 「フック→解説→アドバイス→CTA」の流れ
- コメント欄でよく聞かれていること（次のネタ候補）

**ジャンル別分析：**
- 医療・健康系でバズっているテーマ
- 育児グッズ・便利品でバズっているもの
- 感情共感系でバズっているテーマ
