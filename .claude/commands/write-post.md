### ツリー投稿（（以下コメント欄）の使い方）

内容が多くて1投稿では収まらないとき、または「続きが気になる！」という引きを作りたいときは、
本文の途中に `（以下コメント欄）` を入れてツリー投稿にする。

**ルール：**
- `（以下コメント欄）` より**前**の文章 = Threadsのメイン投稿（200文字前後を目安）
- `（以下コメント欄）` より**後**の文章 = コメント欄への続き投稿（文字数制限なし）
- 区切る場所は「読者が一番続きを読みたくなるところ」で切る。問いかけ・箇条書きの冒頭・驚きの事実の直前など。
- 文の途中（「。」なしで終わる）でもOK。引きが強い場所で切ること。
- **使うかどうかは任意。** 全7本のうち2〜3本に使うのが目安。

**例：**
```
夜に子どもが急変したとき焦ってるママに覚えておいてほしい。

＃8000と＃7119、この2つの使い分けを知らないと、
いざというとき判断が遅れる。

看護師として正直に言うと——

（以下コメント欄）

【＃8000｜小児救急電話相談】
→15歳未満の子ども専用
→夜間・休日（例：19時〜翌8時）に対応
...
```

この場合、`（以下コメント欄）` の前だけがメイン投稿として表示される。

### 全投稿の必須条件
- 文字数：`（以下コメント欄）` がある場合は前後合計で400文字以内を目安。ない場合は200〜350文字。
- 口調：タメ語（〜だよ、〜だよね、〜してみて）
- 1行目：`05_writing.md` のフック型から必ず選ぶ
- 1行目を5案出して、一番強い1つを選ぶ（固有名詞か数字を入れる）
- 構成：フック→解説→アドバイスor商品→CTA
- `07_ng-rules.md` の禁止事項を全て守る
- AI感のある不自然な表現は使わない（「〜と言えるでしょう」等は禁止）
- 同じネタ・同じフックの重複は禁止
- PR投稿は必ず「#PR」「（PR）」を明記する
