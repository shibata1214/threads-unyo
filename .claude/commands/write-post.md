# ライター：投稿文7本自動生成

以下の手順で今日のThreads投稿候補を7本生成してください。

## ステップ1：情報を読み込む
以下のファイルを全て読んでください：
- `01_profile.md`（アカウント概要）
- `02_target.md`（ターゲット像）
- `05_writing.md`（文章ルール・フック型）
- `06_references.md`（バズり構成パターン・競合分析・トレンド）
- `07_ng-rules.md`（禁止事項）
- `next-topics.md`（今週のテーマ）
- `post-history.md`（過去の投稿記録・重複しないように）
- `analysis-latest.md`（最新分析結果・あれば活用）

## ステップ2：7本の投稿文を生成

以下の型を1本ずつ使って7本作成してください：

| # | 型 | 内容の方向性 |
|---|---|---|
| 1 | 権威断言型 | 「看護師として正直に言うと〜」系の医療知識 |
| 2 | 感情物語型 | 「泣ける・忘れられない」系の体験談 |
| 3 | 命に関わる危機感型 | 「知らないと危険・損」系の常識覆し |
| 4 | 季節・時事ネタ型 | 今の時期に刺さるタイムリーな情報 |
| 5 | 否定形注意喚起型 | 「〇〇しないで」「〇〇を買わないで」系 |
| 6 | 共感×先輩ママ型 | 「2人育てて気づいたこと」系のリアル体験 |
| 7 | 商品紹介型（PR） | 楽天・Amazon商品を自然に紹介（PR明記必須） |

### 全投稿の必須条件
- 文字数：200〜350文字
- 口調：タメ語（〜だよ、〜だよね、〜してみて）
- 1行目：`05_writing.md` のフック型から必ず選ぶ
- 構成：フック→解説→アドバイスor商品→CTA
- `07_ng-rules.md` の禁止事項を全て守る
- AI感のある不自然な表現は使わない
- 同じネタ・同じフックの重複は禁止

## ステップ3：post-queue.md に全7本追加

以下のフォーマットで `post-queue.md` に追記してください：

```
---
id: （連番）
型：（上記の型名）
テーマ：（内容のテーマ）
本文：
（投稿文）
予定投稿日時：（未設定）
PR：なし or あり
品質スコア：（自己評価 ★1〜5）
---
```

## ステップ4：結果を表示

- 7本全てを表示する
- 各投稿の「推しポイント」を1行で添える
- はるさんへの手直しポイントがあれば各投稿に1点だけ添える
- 「この中で特に伸びそうな投稿TOP3」を最後に発表する
