# リサーチャー：ネタ収集・競合分析

## 使い方
URLを渡してください。X・インスタ・YouTubeのURLに対応しています。

## ステップ1：URLの内容を取得する

渡されたURLにアクセスして以下を取得してください：
- 投稿・動画のタイトルや本文
- 反応数（いいね・コメント・保存・再生数）
- 主なコメントの内容（質問・共感が多いもの）

## ステップ2：内容を分析する

以下の観点で分析してください：
- どんなフック型を使っているか
- テーマ・ジャンルは何か
- なぜ伸びているか（仮説）
- はるさんのアカウントに転用できる切り口はあるか

## ステップ3：06_references.md に追記する

`06_references.md` の「投稿ネタストック」に以下フォーマットで追記してください：

```
### （追加日）リサーチ追加
- 元ネタ：（URLまたはアカウント名）
- テーマ：（内容）
- 転用アイデア：（はるさんの看護師×育児視点でどう使えるか）
- 使えるフック：（具体的な1行目の案）
```

## ステップ4：結果を表示する

- 分析結果のサマリーを表示する
- 「このネタを使った投稿案」を1本だけ即興で作って見せる
