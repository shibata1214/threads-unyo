# スーパーバイザー：週次レビュー＆改善提案

毎週1回実行して、運用全体を振り返ります。

## ステップ1：全ファイルを確認する
- `post-history.md`（投稿記録と数値）
- `post-queue.md`（待機中の投稿）
- `analysis-latest.md`（最新分析）
- `next-topics.md`（来週のテーマ）
- `07_ng-rules.md`（ルール遵守チェック）

## ステップ2：週次チェックリストを実行する

以下をYES/NOで確認してください：

- [ ] 今日の投稿、誰の何を解決したか説明できるか
- [ ] 投稿が伸びなかった場合、原因を3つ挙げられるか
- [ ] 1週間後のテーマが最低5個あるか
- [ ] 反応が良かった投稿を3回以上擦っているか
- [ ] 有益投稿：PR投稿＝6：4の比率を守れているか
- [ ] 1日3件以上の投稿をしていないか
- [ ] NGルール（薬機法・断定表現・PR未表記）の違反はないか

## ステップ3：改善提案を出す

- 今週上手くいったこと（続けるべきこと）
- 今週上手くいかなかったこと（改善すべきこと）
- 来週試すべき新しいアプローチ1〜3個

## ステップ4：来週の投稿戦略を提案する

- next-topics.md の来週テーマを提案・更新する
- 特に力を入れるべき投稿型を1つ指定する

## ステップ5：レポートを表示する

わかりやすく日本語でまとめて報告してください。
